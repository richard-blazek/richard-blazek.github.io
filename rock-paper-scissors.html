Rock-paper-scissors CA

Rock-paper-scissors is a cellular automaton inspired by the rock-paper-scissors game. A non-empty cell represents a gesture (rock, paper, or scissors). In each step, each cell plays against its four neighbours. If any of the neighbours beats the cell, the cell's gesture will be set to the gesture of that neighbour.

The simulation below uses colours to visualise the gestures of the game. To allow for more complex patterns, it also supports the gestures "lizard" and "Spock" from an extended version of the game.
